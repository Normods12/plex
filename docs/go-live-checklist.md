# Plexonics Go-Live Checklist

**Version 1.0 · Phase 1 Launch**

---

## Infrastructure

- [ ] PostgreSQL production database provisioned and backed up
- [ ] S3 bucket (or Cloudinary account) configured in Strapi for media uploads
- [ ] Strapi deployed to production server (PM2 or Docker), HTTPS enabled
- [ ] Next.js deployed (Vercel recommended or self-hosted with PM2)
- [ ] Nginx/Caddy configured as reverse proxy with SSL (Let's Encrypt)
- [ ] DNS records updated: `www.plexonics.com` → new server IP
- [ ] Old Joomla server kept live for 72h as fallback
- [ ] Environment variables set on all servers (no `.env` files in production)
- [ ] Firewall rules: only ports 80, 443, 22 open externally
- [ ] Strapi admin panel accessible at `/admin` (not publicly indexed)

---

## Content

- [ ] All 8 product domains seeded (run `seed-taxonomy.ts`)
- [ ] Product count matches live site (verify each domain manually)
- [ ] All datasheet PDFs uploaded and linked to products
- [ ] All manual PDFs uploaded and linked
- [ ] All product images present (white background confirmed)
- [ ] `site-settings`: logo files replaced (client to supply SVG + PNG)
- [ ] `contact-info`: address, phone (`1800-1200-023`), mapEmbed confirmed
- [ ] `navigation-menu`: mainNav JSON matches Section 4.1 exactly
- [ ] Alert banner: set `active: false` until ready to use
- [ ] All 8 domains have at least 1 published product

---

## SEO

- [ ] All `/index.php/` redirects tested with `curl -I`
  ```bash
  curl -I https://www.plexonics.com/index.php/products/enterprise-networking/switches
  # Expected: HTTP/1.1 301 Moved Permanently
  # Location: /products/enterprise-networking/switches
  ```
- [ ] PDF redirects tested (pick 5 old Joomla PDF URLs, confirm 301)
- [ ] Google Search Console: submit new sitemap (`/sitemap.xml`)
- [ ] Canonical URLs set on all product pages
- [ ] Meta titles and descriptions populated (spot-check 20 products)
- [ ] `robots.txt` present and correct (allow all, disallow `/admin`)
- [ ] `sitemap.xml` generated by Next.js and accessible
- [ ] Open Graph tags present on homepage and product pages
- [ ] No duplicate content (check canonical tags)

---

## Forms & Email

- [ ] Contact form sends email to `info@plexonics.com` (test in production)
- [ ] hCaptcha or reCAPTCHA v3 active on contact form
- [ ] SMTP relay configured and tested (SendGrid / AWS SES / Mailgun)
- [ ] Product registration form tested end-to-end
- [ ] Email reply-to set correctly (sender's email)
- [ ] Test with real email client (check spam folder)

---

## Performance & Monitoring

- [ ] Lighthouse score > 85 on homepage and a product detail page
  ```bash
  npx lighthouse https://www.plexonics.com --output html --output-path ./lighthouse-report.html
  ```
- [ ] Core Web Vitals: LCP < 2.5s, CLS < 0.1
- [ ] Uptime monitor configured (e.g. UptimeRobot or Better Uptime)
- [ ] Error tracking configured (Sentry on both Strapi and Next.js)
- [ ] Next.js Image Optimization working (check `/api/image` responses)
- [ ] GZIP compression enabled (check `Content-Encoding: gzip` header)
- [ ] Cache headers correct (product images 30d, HTML no-cache)

---

## Security

- [ ] HTTPS enforced (HTTP → HTTPS redirect)
- [ ] HSTS header set
- [ ] Strapi admin JWT secret is strong and unique
- [ ] API tokens rotated from development values
- [ ] No sensitive data in client-side code or public API responses
- [ ] CORS configured to allow only `www.plexonics.com`

---

## Final Checks

- [ ] All pages load without console errors
- [ ] Playwright E2E test suite passes: `npx playwright test`
- [ ] Mobile responsiveness verified on real device (iPhone + Android)
- [ ] Cross-browser check: Chrome, Firefox, Safari, Edge
- [ ] 404 page renders correctly
- [ ] Footer copyright year is current
- [ ] Phone number `1800-1200-023` is clickable on mobile

---

## Post-Launch (within 48h)

- [ ] Monitor Google Search Console for crawl errors
- [ ] Check Sentry for any production errors
- [ ] Verify old Joomla URLs are redirecting correctly in GSC
- [ ] Confirm product images are loading from Strapi media
- [ ] Check server resource usage (CPU, memory, disk)
- [ ] Notify client of successful launch

---

*Checklist covers Plexonics Phase 1 only.*
